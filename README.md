# Кейс 2: Прогнозирование оценки пользователя мобильного приложения

## Описание

Проект по анализу данных и созданию прогнозной модели для предсказания оценки пользователя (1-5) мобильного приложения пенсионного фонда.

**Компания:** Пенсионный фонд  
**Задача:** Цифровая трансформация, повышение удовлетворенности пользователей цифровыми сервисами

## Данные

Датасет `data/app_survey.csv` содержит 42,387 записей с 9 признаками:

- `user_id` — идентификатор пользователя
- `survey_creation_dt` — дата создания опроса
- `survey_response_dt` — дата ответа на опрос
- `language` — язык пользователя
- `age` — возрастная группа
- `gender` — пол
- `tenure_years` — стаж пользователя (годы)
- `user_income` — доход пользователя
- `csat_level` — оценка пользователя (целевая переменная, 1-5)

## Задачи

1. **EDA и инсайты** — анализ данных, выявление закономерностей, бизнес-инсайты
2. **ML-выводы** — аналитические выводы для разработки прогнозной модели
3. **Дашборд** — интерактивный дашборд на Gradio с визуализациями
4. **Презентация** — презентация решения в формате PDF

## Артефакты

- `eda.ipynb` — ноутбук с анализом и визуализациями
- Дашборд на `gradio.app`
- Презентация решения (`.pdf`)
